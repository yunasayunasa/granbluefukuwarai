# HTML修正手順 - アセットブラウザ修正

## 問題
`index.html`の`bottom-panel`に`asset-type-tabs`が存在しないため、アセット一覧が表示されません。

## 手動修正手順

### 修正箇所: `index.html` 136-144行目

**現在の状態:**
```html
  <div class="panel-tools">
    <div class="project-assets" id="asset-list">
      <!-- Asset grid -->
    </div>
  </div>
  <div class="project-footer">
    <input type="range" id="icon-size-slider" min="1" max="3" value="2" />
  </div>
</div>
```

**修正後:**
```html
  <div class="panel-tools">
    <button id="add-asset-button" title="Add Asset">+</button>
  </div>
</div>

<!-- Project View -->
<div id="project-view" class="tab-content">
  <div id="asset-type-tabs" class="asset-tabs">
    <!-- Asset type tabs will be generated by JS -->
  </div>
  <div class="project-assets" id="asset-list">
    <!-- Asset grid -->
  </div>
  <div class="project-footer">
    <input type="range" id="icon-size-slider" min="1" max="3" value="2" />
  </div>
</div>
```

## 変更点

1. `panel-tools`内の`asset-list`を削除し、`+`ボタンを追加
2. `panel-header`の閉じタグ後に`project-view`コンテナを追加
3. `asset-type-tabs`要素を追加（JS がアセットタイプのタブを生成）
4. `asset-list`を`project-view`内に移動
5. `project-footer`を`project-view`内に移動

## 補足
この修正により、`EditorUI.js`の`populateAssetBrowser()`メソッドが正しく動作します。
